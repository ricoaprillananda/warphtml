## Quick Start

```bash
# Clone the repository
git clone https://github.com/your-username/warphtml.git
cd warphtml
```

# Build the Docker image
```
docker build -t warphtml .
```

# Run the container
```
docker run -d -p 8080:80 warphtml
```
